genrl.deep.agents.ppo1.ppo1 module¶

class genrl.deep.agents.ppo1.ppo1.PPO1(network_type, env, timesteps_per_actorbatch=256, gamma=0.99, clip_param=0.2, actor_batch_size=64, epochs=1000, lr_policy=0.001, lr_value=0.001, layers=64, 64, policy_copy_interval=20, tensorboard_log=None, seed=None, render=False, device='cpu', run_num=None, save_model=None, load_model=None, save_interval=50)¶

Bases: object

Proximal Policy Optimization algorithm (Clipped policy).

Paper: https://arxiv.org/abs/1707.06347

Parameters

  • network_type – (str) The deep neural network layer types [‘mlp’]

  • env – (Gym environment) The environment to learn from

  • timesteps_per_actorbatch – (int) timesteps per actor per update

  • gamma – (float) discount factor

  • clip_param – (float) clipping parameter epsilon

  • actor_batchsize – (int) trajectories per optimizer epoch

  • epochs – (int) the optimizer’s number of epochs

  • lr_policy – (float) policy network learning rate

  • lr_value – (float) value network learning rate

  • policy_copy_interval – (int) number of optimizer before copying params from new policy to old policy

  • save_interval – (int) Number of episodes between saves of models

  • tensorboard_log – (str) the log location for tensorboard (if None, no logging)

  • (int) (seed) – seed for torch and gym

  • (str) (device) – device to use for tensor operations; ‘cpu’ for cpu and ‘cuda’ for gpu

  • run_num – (boolean) if model has already been trained

  • save_model – (string) directory the user wants to save models to

  • load_model (string) – model loading path
