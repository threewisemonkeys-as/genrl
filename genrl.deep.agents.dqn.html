class genrl.deep.agents.dqn.dqn.DQN(network_type, env, double_dqn=False, dueling_dqn=False, noisy_dqn=False, categorical_dqn=False, prioritized_replay=False, epochs=100, max_iterations_per_epoch=100, max_ep_len=1000, gamma=0.99, lr=0.001, batch_size=32, replay_size=100, prioritized_replay_alpha=0.6, max_epsilon=1.0, min_epsilon=0.01, epsilon_decay=1000, num_atoms=51, Vmin=- 10, Vmax=10, tensorboard_log=None, seed=None, render=False, device='cpu', save_interval=5000, run_num=None, save_model=None, load_model=None, transform=None)¶

Bases: object

Deep Q Networks

Paper (DQN) https://arxiv.org/pdf/1312.5602.pdf

Paper (Double DQN) https://arxiv.org/abs/1509.06461

Parameters

  • network_type (string) – The deep neural network layer types [‘mlp’, ‘cnn’]

  • env (Gym environment) – The environment to learn from

  • double_dqn (bool) – For training Double DQN

  • dueling_dqn (bool) – For training Dueling DQN

  • noisy_dqn (bool) – For using Noisy Q

  • categorical_dqn (bool) – For using Distributional DQN

  • parameterized_replay (bool) – For using a prioritized buffer

  • epochs (int) – Number of epochs

  • max_iterations_per_epoch (int) – Number of iterations per epoch

  • max_ep_len (int) – Maximum steps per episode

  • gamma (float) – discount factor

  • lr (float) – learing rate for the optimizer

  • batch_size (int) – Update batch size

  • replay_size (int) – Replay memory size

  • tensorboard_log (string) – the log location for tensorboard

  • seed (int) – seed for torch and gym

  • render (bool) – if environment is to be rendered

  • device (string) – device to use for tensor operations; ‘cpu’ for cpu and ‘cuda’ for gpu

  • save_interval (int) – Number of steps between saves of models

  • run_num (int) – model run number if it has already been trained

  • save_model (string) – model save directory

  • load_model (string) – model loading path
