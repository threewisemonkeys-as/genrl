genrl.deep.agents.ddpg.ddpg module¶

class genrl.deep.agents.ddpg.ddpg.DDPG(network_type, env, gamma=0.99, replay_size=1000000, batch_size=100, lr_p=0.0001, lr_q=0.001, polyak=0.995, epochs=100, start_steps=10000, steps_per_epoch=4000, noise=None, noise_std=0.1, max_ep_len=1000, start_update=1000, update_interval=50, layers=32, 32, tensorboard_log=None, seed=None, render=False, device='cpu', run_num=None, save_model=None, load_model=None, save_interval=5000)¶

Bases: object

Deep Deterministic Policy Gradient algorithm (DDPG)

Paper: https://arxiv.org/abs/1509.02971

Parameters

  • network_type (string) – The deep neural network layer types [‘mlp’, ‘cnn’]

  • env (Gym environment) – The environment to learn from

  • gamma (float) – discount factor

  • replay_size (int) – Replay memory size

  • batch_size (int) – Update batch size

  • lr_p (float) – learning rate for policy optimizer

  • lr_q (float) – learning rate for value fn optimizer

  • polyak (float) – polyak averaging weight for target network update

  • epochs (int) – Number of epochs

  • start_steps (int) – Number of exploratory steps at start

  • steps_per_epoch (int) – Number of steps per epoch

  • noise_std (float) – Standard deviation for action noise

  • max_ep_len (int) – Maximum steps per episode

  • start_update (int) – Number of steps before first parameter update

  • update_interval (int) – Number of steps between parameter updates

  • save_interval (int) – Number of steps between saves of models

  • layers (tuple) – Number of neurons in hidden layers

  • tensorboard_log (string) – the log location for tensorboard

  • seed (int) – seed for torch and gym

  • render (bool) – if environment is to be rendered

  • device (string) – device to use for tensor operations; [‘cpu’,’cuda’]

  • run_num (int) – model run number if it has already been trained

  • save_model (string) – model save directory

  • load_model (string) – model loading path
